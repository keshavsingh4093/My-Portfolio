Link Shrinker

Link Shrinker is a web application that allows users to generate short URLs for long URLs. It also keeps track of previously generated URLs, which users can view on the "Previous URLs" page. The application provides a seamless user experience by maintaining login sessions for 7 days.

  • Developed a URL shortener application that converts long URLs into shorter, shareable links.
  • Enabled users to view previously created short URLs on a dedicated "Previous URLs" page.
  • Implemented secure user registration and login with a 7-day session management feature for seamless user experience.
  • Built backend logic using RESTful APIs to manage URL generation and storage.
  • Designed a clean, responsive interface for smooth user interaction.

Tech Stack: Node | Express | MongoDB | HTML | Tailwind
